# HooBank - Modern UI/UX website using React.js & Tailwind CSS
A modern and responsive banking website built with React, Vite, and Tailwind CSS. 
The project features a sleek landing page with interactive UI components, smooth layouts, and a mobile-friendly design, showcasing a digital banking experience.

Technologies Used:
React.js
Vite
Tailwind CSS
HTML5
JavaScript (ES6+)
CSS3
PostCSS
Autoprefixer
